#  StringStack Talent OS

A complete Training, Learning and Placement Management System built using Java, Spring Boot, React, MySQL and JWT Authentication.

---

##  Project Overview

StringStack Talent OS is an enterprise-level web application developed for training institutes to manage students, trainers, courses, batches, mock interviews and placements through a single platform.

---

##  Objectives

- Student Management
- Trainer Management
- Course Management
- Batch Management
- Attendance Management
- Mock Interviews
- Placement Tracking
- Resume Management
- Dashboard & Reports
