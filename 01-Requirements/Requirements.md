# StringStack Talent OS

## Project Overview

StringStack Talent OS is a web-based Training, Learning, and Placement Management System designed for training institutes like StringStack.Ai.

The system helps manage students, trainers, courses, batches, mock interviews, attendance, placements, and company hiring through a single platform.

---

# Project Objectives

- Digitalize the complete training process.
- Reduce manual work using spreadsheets.
- Track student learning progress.
- Manage mock interviews and feedback.
- Simplify placement activities.
- Provide separate dashboards for Admin, Trainer, Student, and HR.
- Generate reports and analytics.

---

# User Roles

## 1. Admin

### Responsibilities

- Manage the entire application
- Create and manage Trainers
- Create and manage Students
- Create Courses
- Create Batches
- Assign Trainers to Batches
- Assign Students to Batches
- Manage Companies
- View Placement Reports
- View Dashboard Analytics
- Manage User Roles
- Reset User Passwords
- Activate/Deactivate Users

---

## 2. Trainer

### Responsibilities

- Login
- View Assigned Batches
- View Assigned Students
- Mark Attendance
- Upload Study Materials
- Conduct Mock Interviews
- Add Interview Feedback
- Add Student Performance Remarks
- Update Student Progress
- View Course Details

---

## 3. Student

### Responsibilities

- Register/Login
- Update Profile
- Upload Resume
- View Assigned Batch
- View Course Materials
- View Attendance
- View Mock Interview Feedback
- View Performance Report
- Apply for Placement Drives
- View Placement Status
- Download Certificates (Future)

---

## 4. HR

### Responsibilities

- Login
- Search Students
- View Student Profiles
- Download Resume
- Schedule Interviews
- Add Interview Feedback
- Select/Reject Candidates
- Update Placement Status
